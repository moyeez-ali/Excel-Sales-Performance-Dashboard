# Excel Sales Performance Dashboard

## Project Overview

This project is an interactive **Sales Performance Dashboard** built in Microsoft Excel as part of my Data Analytics learning journey. The dashboard provides valuable business insights by analyzing sales performance across different categories, states, customers, and time periods.

The project demonstrates the complete workflow of a Data Analyst—from importing and preparing data to creating meaningful visualizations and dashboards that support business decision-making.

---

## Objectives

* Analyze overall sales performance.
* Identify top-performing product categories.
* Monitor monthly sales trends.
* Evaluate yearly profit performance.
* Discover top customers based on sales.
* Compare sales across different states.
* Present business insights through an interactive dashboard.

## Dashboard Features

The dashboard includes:

* Sales by Category
* Monthly Sales Analysis
* Profit by Year
* Top 5 Customers
* Sales by State
* Customer Count
* Interactive Dashboard for business insights

---

## Project Workflow

1. Imported the sales dataset into Excel.
2. Prepared the dataset for analysis.
3. Created Pivot Tables for business reporting.
4. Built Pivot Charts for visual analysis.
5. Designed an interactive Excel dashboard.
6. Generated business insights from the data.

---

## Key Business Questions Answered

* Which product categories generate the highest sales?
* How do sales change month by month?
* Which year generated the highest profit?
* Who are the top five customers?
* Which states contribute the most to sales?
* How many unique customers are included in the dataset?
